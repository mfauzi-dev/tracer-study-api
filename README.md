# Tracer Study Backend

## **Clone repository**

```bash
git clone https://github.com/mfauzi-dev/tracer-study-backend
```

## **Setup Project Baru**

1. **Rename folder sesuai project baru**

    ```bash
    mv tracer-study-backend my-new-project
    cd my-new-project
    ```

2. **Hapus riwayat Git lama**

    ```bash
    rm -rf .git
    ```

3. **Install dependencies**

    ```bash
    npm install
    ```

4. **Buat file .env berdasarkan .env.example**

    ```bash
    cp .env.example .env
    ```

5. **Buat folder untuk upload**

    ```bash
     mkdir uploads
    ```

6. **Jalankan project**

    ```bash
    npm run dev
    ```
